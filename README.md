# Materialize Angular ArcGIS Widgets
Angular directives to build Esri application in material design. Following directive are available:

- Map
- ArcGISDynamicMapServiceLayer
- FeatureLayer
- BasemapGallery
- DrawGraphics
- Directions
- Search
- Legend 
- Layer
